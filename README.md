# About

PastaBrain is a Vercel web application with pasta quiz. 🍝 🧪

## Tech stack

The project contains the following tech stack:

- NextJS
- TailwindCSS
- Biome
- Vercel
- Husky

## Setup

Clone repository:

```bash
git@github.com:sarkaaa/pastabrain.git
```

Install dependencies:

```bash
pnpm dev
```

Run project locally:

```bash
pnpm dev
```

Build project:
```bash
pnpm build
```
